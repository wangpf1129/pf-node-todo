# 一个可以在命令行使用的todo
## 下载
下载npm包：
```bash
npm install -g pf-node-todo
// 或者
yarn global add pf-node-todo
```
下载后可查看版本：
```bash
todo --version 
```
## 如何使用
1. todo   查看所有任务列表
    - 能够操作所有增删改查
2. todo add <taskName> 添加一个任务
3. todo clear 清空所有任务
